GitHub Pages とは

GitHub Pages は、あなたの プロジェクト、ポートフォリオ、 ドキュメント などを簡単に公開できる 無料ホスティングサービス です。

このページも GitHub Pages で公開されています！

使い方

GitHub Pages はとても簡単に使えます：

  1. リポジトリを作成する
  2. HTML, CSS, JavaScript ファイルを追加
  3. GitHub Pages 設定を有効化
  4. サイトが公開される！

詳しくは GitHub のドキュメントをご覧ください
